### Como usar Uso de permissão ao script
```sh
chmod +x istalapp.sh
```
Agora execute o script
```sh
./instalapp.sh
```
E só espere ate que tudo seja instalado este script atualiliza o linux primeiro por tanto o tempo vai depender da sua intenet .
